## 评估指标说明

| 指标 | 说明 |
|------|------|
| Recall@K | 检索结果中包含正确答案的比例 |
| MRR | 第一个相关结果排名的倒数平均 |
| NDCG | 归一化折损累计增益 |
| Answer Accuracy | 答案的事实正确性 |
| Citation Precision | 引用文档的准确性 |
